Fun Hobby Projects

I've not pursued many hobby projects, mainly due to my school/work schedule being so full for the past 5 years. Hoping to tackle more after I graduate. Nonetheless, there are 2 notable projects which I began:

  • RallyCats Spelling Error Automated Response Python Script
    • At UC, I was part of an athletics club called RallyCats which maintained a points system for ranking member involvement. The leading point earners at the end of each semester won some fancy prizes like jerseys, basketballs, etc. One of the easiest ways to earn points was by finding the spelling error in the club's weekly newletter emails. The first member to locate the spelling error & return an email to the club with the error indicated would receive 5 free points.
    • Well, after being beaten one week—when I'd returned an email within 2 minutes—I decided to take a true Software Developer's approach and built an automated email parsing & returning script in Python. Essentially, upon receiving an email from the club, my script would automatically parse & locate the spelling error, create a new email & populate it with the located spelling error, and then send it off to the club. This entire process took a matter of seconds, rending my solution far more effective than that of my fellow members. I had to do a bunch of tuning on the spelling error algorithm to ignore names, and supplied it a list of UC-related terms & phrases which were also to be ignored.
    • Unfortunately, the script only got to see ~8 weeks of use before my email got blacklisted as a possible bot (from the club's email, NOT the university's). I began the process of porting the project from my university MS Outlook account to my personal Gmail, but never finished the port as school picked up pace.
  • Instagram Comment Bot for Entering Lightsaber Giveaways
    • This project was even less complete than the RallyCats email script, but also completed in Python. In early 2024, I was interested in winning a ~$500 lightsaber giveaway from a popular lightsaber manufacturer on Instagram. Each comment—tagging 3 new people—would earn you 1 entry into the giveaway. My plan was to create a simple Python script that went through the account's list of followers, taking 3 at a time and commenting them on the post. It would continue running until the account's entire list of followers (~20k, or ~6500 entries) had been traversed.
    • I did some brief research & planning for the project, found a couple useful Python modules to enable communication with Instagram's API's, and even wrote a simple script to run everything. However, unable to find a clear line as to what rate I could run the script at before getting banned for spam, I ultimately left the project unfinished. I may yet deploy this project in the future—perhaps with a secondary Instagram account rather than my personal—as the company has continued running monthly giveaways ever since.

Due to the rather informal & incomplete nature of these projects, I never threw polished code up on GitHub (also, for the RallyCats project, I rather simply hard-coded my credentials into a text file). However, code can be viewed upon request as I've still got the projects saved.
